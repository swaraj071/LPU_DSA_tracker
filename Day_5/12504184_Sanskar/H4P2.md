# H4P2 - Find Numbers with Even Number of Digits

- https://leetcode.com/problems/find-numbers-with-even-number-of-digits/submissions/2046651987/

```python
class Solution:
    def findNumbers(self, nums: List[int]) -> int:
        count=0
        for i in range(0,len(nums)):
            if len(str(nums[i]))%2==0:
                count+=1
        
        return count
```
